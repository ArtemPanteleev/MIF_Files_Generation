# MIF Files Generation

Qt-приложение для подготовки и генерации файлов в формате **Memory Initialization File (MIF)**, используемом при описании ПЗУ/ПЛИС. Инструмент помогает:
- сконвертировать произвольные числовые выборки из CSV/текстовых файлов в корректный MIF с нужной шириной, глубиной и системами счисления;
- синтезировать тестовые паттерны MIF на основе заданных формул (арифметические операции, тригонометрия, степень, корень и т.д.);
- гибко управлять разрядностью адреса/данных, диапазонами, типом исходных данных и форматом результата.

## Возможности

- **Конвертер MIF**  
  Загружает исходные значения, нормирует их к заданной разрядности и сохраняет в формате MIF. Отдельно задаются форматы адреса (`BIN`, `OCT`, `UNS`, `HEX`) и данных, а также система счисления входного файла (bin/oct/dec/hex).

- **Генератор паттернов**  
  Создаёт синтетический контент MIF по параметрам `WIDTH`, `DEPTH`, диапазону значений и выбранной операции (`ADDITION`, `SUBTRACTION`, `MULTIPLICATION`, `DIVISION`, `DEGREE`, `RADICAL`, `SINUS`, `COSINE`, `ZERO`). Позволяет быстро наполнить память типовыми тестами.

- **Сохранение пресетов**  
  Все настройки хранятся через синглтон `Settings`, поэтому переключение между режимами не сбрасывает введённые параметры.

- **Интерфейс Qt**  
  Удобная форма `mainwindow.ui` с подсказками и индикаторами состояния: блоки выбора файла, настроек и статуса генерации.

## Стек и требования

- Qt 5.10+ (проект создан под Qt Creator, qmake, MinGW 32-bit)
- C++11 (стандарт библиотек Qt)
- Windows (проект собирался под Win32; кроссплатформенность зависит от доступности Qt toolchain)

## Сборка

### Qt Creator (qmake)
1. Открой `NIR/NIR.pro` в Qt Creator.
2. Выбери комплект сборки (например, Desktop Qt 5.10 MinGW 32-bit).
3. Нажми *Configure Project*, затем *Run qmake* и *Build*.
4. Запусти приложение кнопкой *Run*.

### Командная строка
```bash
cd NIR
qmake NIR.pro
mingw32-make
```

Исполняемый файл появится в каталоге `build-*/{debug,release}/NIR.exe`.

## Использование

### Конвертация готовых данных
1. Нажми **Открыть** и выбери исходный файл (значения разделены запятыми или переводами строк).
2. Укажи систему счисления входных данных (блок *Start*).
3. Настрой `WIDTH`, `DEPTH`, формат адреса и данных, затем **Применить**.
4. Выбери путь сохранения (**Сохранить**).
5. Нажми **Получить** — появится готовый MIF.

### Генерация паттернов
1. Задай `WIDTH`, `DEPTH`, адрес/данные для секции генерации и нажми **Применить**.
2. Внеси параметры диапазона (`MIN`, `MAX`), множители (`NUMBER`, `NUMBER_2`) и тип операции.
3. Укажи путь сохранения результата.
4. Нажми **Сгенерировать** — файл создастся автоматически.

Статусы операций отображаются в полях `INFORMATION` и `INFORMATION_GENERATION`.

## Советы

- Для больших глубин необходимо следить, чтобы входные данные содержали не меньше значений, чем `DEPTH`, иначе остаток памяти не будет заполнен.
- При выборе `BIN` для данных генератор вручную формирует двоичные строки, поэтому `WIDTH` должен соответствовать реальной шине данных.
- Повторяющиеся настройки удобно хранить, просто меняя входной/выходной файл — остальные параметры сохранятся.
